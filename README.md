# Data-insta
App simple para compartir imagenes.

Creado con JavaScript y NodeJs. Base de datos nedb.

A través de esta simple aplicación puedes aprender a crear un proyecto sencillo con NodeJS abarcando lado cliente, lado servidor y utilizando una base de datos sencilla basada en MongoDB llamada nedb.
